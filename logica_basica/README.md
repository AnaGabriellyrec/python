# Projeto DIO

Projetos desenvolvidos durante meus estudos em Python, com foco em lógica de programação e resolução de problemas práticos.

---

## Projetos

### Lógica Básica

#### Sistema de Promoções

Sistema que analisa o valor de uma compra e define benefícios (mensagem, brinde ou desconto) com base em condições.

#### Validação de Transação Bancária

Valida transações a partir de valor, taxa e mínimo permitido, decidindo se a operação é aprovada.

#### Padronização de Nomes

Tratamento de strings para remover espaços, ajustar letras e padronizar nomes.

---

### Projetos Interativos

#### Controle Financeiro

Sistema que permite registrar ganhos e gastos, exibindo o saldo atual com base nos valores informados pelo usuário.

#### Sistema de Estoque

Sistema de gerenciamento de produtos que permite adicionar, remover e listar itens, com validações básicas.

---

## Conceitos praticados

* Estruturas condicionais (`if`, `elif`, `else`)
* Estruturas de repetição (`while`, `for`)
* Manipulação de listas
* Manipulação de strings
* Operadores matemáticos e comparações
* Entrada de dados (`input`)

## Organização do Repositório

Este repositório está dividido em duas pastas para organizar melhor os projetos de acordo com o nível e tipo de aprendizado.

### Lógica Básica

Nesta pasta estão projetos mais simples, focados em fundamentos da programação, como:

* Estruturas condicionais
* Operadores
* Manipulação de strings

São exercícios voltados para o desenvolvimento da lógica e resolução de problemas iniciais.

### Projetos Interativos

Aqui estão projetos mais completos, com interação com o usuário via terminal.

Nessa pasta você encontrará:

* Sistemas com menus interativos
* Uso de listas para armazenamento de dados
* Estruturas de repetição e validações

Esses projetos simulam situações mais próximas de aplicações reais, mostrando evolução na construção de sistemas.

---

Essa organização foi feita para demonstrar de forma clara a progressão do aprendizado, desde conceitos básicos até aplicações mais estruturadas.

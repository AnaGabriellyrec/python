# Projeto DIO

Esses projetos foram desenvolvidos durante meus estudos em Python, com foco em lógica de programação 
e resolução de problemas práticos. Cada um deles trabalha conceitos importantes como entrada de dados,
estruturas condicionais e manipulação de strings.

---

## Sistema de Promoções

Nesse projeto, criei um sistema que analisa o valor total de uma compra e define automaticamente qual benefício o cliente recebe.

Dependendo do valor:
- Pode receber apenas uma mensagem de agradecimento
- Pode ganhar um brinde
- Ou receber descontos diferentes

Esse projeto me ajudou a entender melhor como usar estruturas condicionais (`if`, `elif`, `else`) e trabalhar com diferentes faixas de valores.

---

## Validação de Transação Bancária

Aqui desenvolvi uma lógica para validar transações financeiras.

O sistema:
- Recebe o valor da transação, uma taxa e um valor mínimo
- Calcula o valor final (valor - taxa)
- Verifica se a transação pode ser aprovada ou não

Esse projeto foi importante para praticar operadores matemáticos e comparações, além de entender melhor como tomar decisões no código.

---

## Padronização de Nomes

Nesse projeto, trabalhei com manipulação de strings.

O objetivo foi:
- Remover espaços extras
- Corrigir letras maiúsculas e minúsculas
- Garantir que cada palavra comece com letra maiúscula

Esse exercício me ajudou a entender melhor funções como `.strip()`, `.split()` e `.title()`.

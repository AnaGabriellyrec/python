# Site de Aniversário

Site desenvolvido com HTML, CSS e JavaScript, focado em interatividade, animações e personalização visual.

## Tecnologias:

- HTML5
- CSS3
- JavaScript

## Funcionalidades:

- Layout responsivo
- Animações e efeitos visuais
- Interações com JavaScript
- Exibição de imagens e mensagens


## Objetivo:
Praticar desenvolvimento Front-End e criar uma página interativa e personalizada.
